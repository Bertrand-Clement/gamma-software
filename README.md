# GAMMA-SOFTWARE

Requirements
------------
* Vagrant
* VirtualBox
* Make

Installation
------------
```bash
$ make init
```


Add a line in your hosts file :
```bash
192.168.40.98   gamma-software.local.com
```

Run the project :
```bash
$ make start
```

View the project :
- https://gamma-software.local.com
- https://gamma-software.local.com/phpmyadmin

Stop the project :
```bash
$ make stop
```

BDD
------------
gamma-software.sql is on the main folder
